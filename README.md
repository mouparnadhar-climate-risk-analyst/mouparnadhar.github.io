Bridging the gap between Geospatial Science, Environmental Data, and Financial Risk Management.

This repository contains the source code for my professional portfolio website. It is designed to showcase my technical expertise in ESG Data Analytics, Geospatial Intelligence, and Climate Risk Modeling.

🚀 Project Overview

This portfolio is a high-performance, single-page application (SPA) built to demonstrate how complex climate and environmental datasets can be transformed into actionable, boardroom-ready intelligence.

The site serves as a live demonstration of my ability to combine modern frontend engineering (React/Tailwind) with advanced analytical domains (GIS/Remote Sensing/ESG).

💎 Key Features

Interactive Case Studies: Dynamic filtering of projects by category (Institutional vs. Academic) using Alpine.js.

Modern UI/UX: A "Deep-Tech" aesthetic featuring Glassmorphism, smooth transitions, and a responsive design optimized for all devices.

Data-Driven Design: Visual representations of key performance metrics (e.g., Stranded Asset Risk, Ecosystem Monitoring hectares).

SEO & Performance Optimized: Built for lightning-fast load times and high visibility.

🛠️ Technical Stack
Frontend & Web Engineering

Framework: HTML5 / Tailwind CSS (Utility-first styling)

Interactivity: Alpine.js (Lightweight reactive component framework)

Icons: FontAwesome 6

Fonts: Inter (UI) & JetBrains Mono (Data/Technical)

Domain Expertise (Demonstrated via Projects)

Geospatial: Google Earth Engine (GEE), QGIS, Sentinel-1/2, Copernicus GLO-30.

Data Science: Python, SQL (Spatial Joins), Predictive Modeling, MCDA.

ESG Frameworks: CSRD, TCFD, ESRS, IPCC AR6.

📂 Featured Case Studies

The portfolio highlights five core pillars of my work:

TerraQuant Platform: A SaaS solution for real-estate climate stress-testing.

Alutaguse Peatland Monitor: An EO Digital Twin for carbon liability simulation.

Palm Jumeirah Index 2050: High-resolution stranded asset risk modeling.

MBR Solar Park Optimization: Geospatial capital allocation for renewable energy.

Master's Dissertation: Importance-Performance Analysis (IPA) for public infrastructure.
